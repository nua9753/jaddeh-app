# جاده

اپ نصب‌شوندهٔ برنامه هشت‌هفته‌ای رانندگی مستقل، ثبت میزان ترس، چک‌لیست پیش از حرکت و تمرین تنفس.

## اجرای محلی

```bash
npm install
npm run dev
```

## انتشار در GitHub Pages

پوشه را در یک مخزن GitHub با شاخهٔ `main` قرار دهید. سپس در تنظیمات مخزن، از بخش **Pages**، گزینهٔ **Source** را روی **GitHub Actions** بگذارید. با هر Push، نسخهٔ جدید خودکار ساخته و منتشر می‌شود.

اطلاعات شخصی اپ فقط در مرورگر همان دستگاه ذخیره می‌شوند.

## ساخت APK اندروید بدون Android Studio

گردش‌کار `Build Android APK` با هر Push روی شاخهٔ `main` اجرا می‌شود. برای دریافت فایل:

1. وارد صفحهٔ مخزن در GitHub شوید.
2. تب **Actions** را باز کنید.
3. آخرین اجرای **Build Android APK** را انتخاب کنید.
4. از بخش **Artifacts** فایل `jaddeh-android-apk` را دریافت کنید.
5. فایل ZIP را باز و `app-debug.apk` را روی گوشی اندرویدی نصب کنید.

این APK برای نصب مستقیم و آزمایش است. انتشار در Google Play به کلید امضا و خروجی Release/AAB نیاز دارد.
